# Commands

## Generate output file for a particular program (xxx.s/c)

```bash
make xxx.out
```

## Open visual debugger for a particular program (xxx.s/c)

This command will firstly call `make xxx.out`, and then run the visual debugger automatically.

```bash
make xxx.vis
```
